**Special: hex-tinted active chip on /shop (D-15).** When customer selects a chip, the pill background becomes the colour's actual hex, alpha-blended with white for legibility. Pure-black / pure-white edge cases are handled by the contrast helper described in **Color Math Helper** below.

### Color math helper (NEW, source: `ui-ux-pro-max` accessibility patterns + WCAG 2.2 SC 1.4.11)

Required because D-15 says "active state = pill background tinted with hex (alpha-mixed for WCAG against text — fallback to dark text on light hex)."

```ts
// src/lib/colour-contrast.ts (new file — Phase 18 plan)
export function getReadableTextOn(hex: string): "#FFFFFF" | "#0B1020" {
  // sRGB relative luminance per WCAG 2.2.
  const c = hex.replace("#", "");
  const r = parseInt(c.slice(0, 2), 16) / 255;
  const g = parseInt(c.slice(2, 4), 16) / 255;
  const b = parseInt(c.slice(4, 6), 16) / 255;
  const lin = (v: number) => (v <= 0.03928 ? v / 12.92 : Math.pow((v + 0.055) / 1.055, 2.4));
  const L = 0.2126 * lin(r) + 0.7152 * lin(g) + 0.0722 * lin(b);
  // Threshold 0.5 keeps Bambu Sunset Orange + Polymaker Lime green safely on dark text;
  // pitch black + ink-near tones flip to white. Validated mentally against the seeded set.
  return L > 0.5 ? "#0B1020" : "#FFFFFF";
}
```

This helper is the SINGLE source of truth for "what text colour goes on a hex pill."
