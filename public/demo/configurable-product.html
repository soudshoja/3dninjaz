Locked decisions

Updated picks after this round

1 · Field types in v1

✓ Ship all 4: text · number · colour · select. Future-proof, ~1h extra.

2 · Pricing

✓ Price tier table — admin sets fixed price per letter count (1=MYR 7, 2=MYR 9, 3=MYR 12 …). Lookup, not multiplication. Admin sets max letter count.

3 · Text validation

✓ Per-field allowedChars regex + global profanity allowlist (small word list, editable later).

4 · Stock model

✓ No per-row stock. isActive on/off only. Print-on-demand.

5 · Colour subset

✓ Admin curates colours per field. Customer only sees the chosen subset, not the full library. Each colour field stores allowed colour ids.

6 · Image gallery

✓ No image limit. Admin writes caption per image. Server compresses (Sharp → WebP/AVIF + multi-resolution srcset) for fast LCP.

7 · Preview component

✓ Generic styled-SVG name strip. Colour-aware. PDP swaps display image → live preview when customer types/picks.

8 · Cart dedupe

✓ Same JSON config → qty bump. Different → new line.

9 · Backwards compat

✓ All existing products auto-set to productType='stocked'. Variant editor stays default for them.
